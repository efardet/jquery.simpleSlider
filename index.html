jQuery Simple Slider

jQuery Simple Slider is a simple, lightweight and "expandable" jQuery plugin with several options.
In order to work, it may be applied to a 'wrapper' div containing the elements to slide.

Example of simple integration : jQuery(".slider1").simpleSlider
Note: You can use ID or class.

The plugin has options to set/unset navigation elements, and defining columns and slide speed :

navigation: display/hide Prev/Next buttons. Set to true or false
prevText: Set text for previous nav button. Use with quotes ("")
nextText: Same as above, for next nav button
pagination: Used to display/hide numbered navigation. Set to true or false
autoplay: Auto slide items. Set to true or false
delay: Time in milliseconds between transitions
speed: Slide speed in milliseconds
numItems: Display items in columns. Default : 1

I have added the touchwipe support for mobile devices, this is optional of course for the plugin to work.

Code

jQuery(".slider6").simpleSlider({
	navigation: true,
	prevText: "Previous",
	nextText: "Next",
	pagination: true,
	autoplay: true,
	delay: 8000,
	speed: 500,
	numItems: 3
});
